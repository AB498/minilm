MiniLM Text Embedding Service

A serverless API that provides text embeddings and FAQ search using the all-MiniLM-L6-v2 model.

API Endpoints

POST /api/embed

Generate embeddings for one or more texts.

Request:

{
  "texts": ["This is an example sentence", "Each sentence is converted"]
}

Response:

{
  "embeddings": [[...384 values...], [...384 values...]],
  "dimensions": 384,
  "processing_time_seconds": 0.123,
  "texts_processed": 2
}

POST /api/faq

Find the most relevant FAQ matches for a query.

Request:

{
  "query": "How do I donate?",
  "top_k": 3
}

Response:

{
  "matches": [
    {
      "question": "How can I make a donation",
      "answer": "You can donate through our app using mobile banking...",
      "score": 0.8976
    },
    ...
  ],
  "processing_time_seconds": 0.045,
  "query": "How do I donate?"
}

GET /api/info

Get information about the model.

Response:

{
  "model_name": "sentence-transformers/all-MiniLM-L6-v2",
  "embedding_dimensions": 384,
  "max_sequence_length": 128
}

Model Information

This service uses the all-MiniLM-L6-v2 model from the sentence-transformers library, implemented with JavaScript using the Xenova/transformers library.
